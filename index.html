Polymer-element (блок)

			Polymer('my-element', {
			    created: function() { ... },
			    ready: function() { ... },
			    detached: function() { ... },
			    attributeChanged: function(attr, oldVal, newVal) {
			        console.log(attr, 'old: ' + oldVal, 'new:', newVal);
			    }
			});

На самом деле

document.registerElement('my-element');

Элемент внутри

			<template>
			  <!-- здесь верстка -->
			  <content></content>
			</template>

Возможности в template

			{{items.length}} элемента
			<template repeat="{{item in items}}">
			    {{item.title}}
			</template>

Больше в документации

Функция, которая творит чудеса

element.createShadowRoot();

Shadow DOM защитит стили элемента

			<style>
			<my-element>
			    #shadow-root
			        <style>
			        <p>этот текст в Shadow DOM<p>
			    <p>а этот абзац снаружи<p>
			</my-element>

Собираем все вместе

			<polymer-element name="my-element">
			    <template>
			    </template>
			    <script>Polymer({})</script>
			</polymer-element>

Переиспользование

С помощью HTML-imports

			<link rel="import"
			  href="bower_components/polymer-ymaps/polymer-ymaps.html"/>

Устанавливаем через bower

			bower install polymer-ymaps

Тестирование

npm install web-component-tester

  • Встроенные mocha+chai
  • Тесты пишутся на html (можно и на js)
  • Без PhantomJS, он не поддерживает новые технологии
  • Зато с SauceLabs или другим Selenium Grid

Блоки

Here will be couple of blocks.

Polymer-element (блок)

			Polymer('my-element', {
			    created: function() { ... },
			    ready: function() { ... },
			    detached: function() { ... },
			    attributeChanged: function(attrName, oldVal, newVal) {
			         console.log(attrName, 'old: ' + oldVal, 'new:', newVal);
			    },
			});

Элемент имеет содержимое

			<template>
			  <!-- здесь верстка -->
			  <content></content>
			</template>

Shadow DOM защитит стили элемента

			<style>
			<my-element>
			    #shadow-root
			        <style>
			        <p>этот текст в Shadow DOM<p>
			    <p>а этот абзац снаружи<p>
			</my-element>

Возможности в template

			{{items.length}} элемента
			<template repeat="{{item in items}}">
			    {{item.title}}
			</template>

Тестирование

  • Встроенные mocha+chai
  • Тесты пишутся как html (можно и на js)
  • Без PhantomJS, он не поддерживает новые технологии
  • Зато с SauceLabs или другим Selenium Grid
